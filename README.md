# Melt Companion Plugin (octoprint-melt)

This OctoPrint plugin serves as the high-performance backend telemetry engine for the Melt Flutter application. It fundamentally shifts the computational burden away from the OctoPrint Web UI and aggregates data for native mobile and desktop clients.

## Key Features
- **MessagePack WebSockets:** Broadcasts real-time printer telemetry (`PrintProgress`, `Temperatures`, `ZChange`) using highly compressed `msgpack` payloads, eliminating JSON serialization bottlenecks on the Raspberry Pi.
- **Cancel Object Safety:** Intercepts `plugin_cancelobject_cancel` events. If a user skips the final object in a queue, this plugin automatically injects safety G-code to shut down the heaters and park the toolhead, preventing thermal runaway.
- **G-Code Thumbnail Extraction:** Exposes a lightning-fast REST endpoint to parse and serve Base64 encoded thumbnails generated by PrusaSlicer/Cura directly from the `.gcode` file.
- **Deep Telemetry Analytics:** Logs all print jobs and timeseries temperature data natively in a local `metrics.db` SQLite database.

## REST API Endpoints

All endpoints are authenticated and require an OctoPrint Application Key.

| Endpoint | Method | Description |
| :--- | :--- | :--- |
| `/plugin/melt/telemetry` | `GET` | Returns aggregated state, temperatures, and 3rd party plugin data. |
| `/plugin/melt/mesh` | `GET` | Relays the 3D topography matrix from `BedLevelVisualizer`. |
| `/plugin/melt/thumbnail/<filename>` | `GET` | Returns the parsed PNG thumbnail image from the G-Code header. |
| `/plugin/melt/toggle/<plugin_name>` | `POST` | Toggles optimization plugins (e.g., `arc_welder`, `port_retry`). |

## WebSocket Protocol

Clients should listen to the OctoPrint plugin message channel for the identifier `melt`.
Payloads are **MessagePack** encoded binaries.

**Example Decoded Schema:**
```json
{
  "type": "telemetry_update",
  "event": "PrintProgress",
  "payload": { "completion": 45.2 },
  "current_data": { ... },
  "plugin_data": { "spool_manager": "available" }
}
```
